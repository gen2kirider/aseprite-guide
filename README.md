# Aseprite 日本語ガイド

Asepriteの使い方を日本語で学べる、非公式のファンメイドガイドです。
基本操作からアニメーション、パレット運用、書き出しまで、実制作で使う内容をカテゴリ別にまとめています。

## サイトはこちら

https://aseprite-guide.vercel.app/

## このサービスでできること

- 初心者向けにAsepriteの基本操作を学べる
- 用語図鑑で制作中に用語をすぐ確認できる
- カテゴリ別の記事で必要な機能だけ素早く探せる
- 実践的な小技やワークフローを日本語で確認できる
