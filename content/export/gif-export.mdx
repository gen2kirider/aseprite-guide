## GIF書き出しの基本

<Term id="pixel-art">ドット絵</Term>
アニメーションをSNSやWebサイトで共有するなら、GIF形式が最も手軽です。AsepriteはGIF書き出しに対応しており、細かい設定も可能です。

## 書き出し手順

### 基本の書き出し

1. **File > Export As**（`Ctrl+Shift+E` でなく `Ctrl+Alt+Shift+S`）
2. ファイル名の拡張子を `.gif` にする
3. 設定を調整
4. **Export** をクリック

> **Save As** ではなく **Export As** を使います。Save Asだとaseprite形式で保存されます。

### GIF設定ダイアログ

書き出し時に以下の設定が表示されます：

| 設定           | 説明           | おすすめ         |
| -------------- | -------------- | ---------------- |
| Pixel Scale    | 拡大倍率       | 2x〜4x           |
| Animation Loop | ループ回数     | Infinite（無限） |
| Interlaced     | インターレース | OFF              |

## 拡大して書き出す

ドット絵は原寸だと小さすぎてSNSでは見づらいです。**Pixel Scale** で拡大しましょう。

| プラットフォーム | おすすめサイズ | Pixel Scale例（32×32の場合） |
| ---------------- | -------------- | ---------------------------- |
| Twitter/X        | 256〜512px     | 8x〜16x                      |
| Instagram        | 512〜1080px    | 16x〜32x                     |
| Discord          | 128〜256px     | 4x〜8x                       |
| Webサイト        | 自由           | 2x〜4x                       |

> 拡大は必ず **整数倍** で。Pixel Scaleが整数倍なら、ドットがきれいに保たれます。

### Sprite Size で事前拡大

Export Asの Pixel Scale ではなく、事前にリサイズする方法もあります：

1. **Sprite > Sprite Size**
2. 倍率を指定（200%, 400% 等）
3. **Interpolation: Nearest Neighbor** を選択
4. リサイズ後に Export As で書き出し

## 色数とファイルサイズ

GIFは**最大256色**の制限があります。

### 色数が多い場合

RGBAモードで多くの色を使っている場合、256色に減色されます。ドット絵は元々色数が少ないので、ほとんどの場合問題ありません。

### ファイルサイズを小さくするには

| 方法                             | 効果       |
| -------------------------------- | ---------- |
| フレーム数を減らす               | 大幅に削減 |
| 画像サイズを小さく               | 大幅に削減 |
| 色数を減らす                     | やや削減   |
| アニメーションの動く範囲を小さく | やや削減   |

> Twitter/Xの GIF上限は **15MB**。ドット絵なら通常問題ありませんが、大きなキャンバスの長いアニメーションでは注意。

## 背景の処理

### 透明背景

GIFは**透明をサポート**しますが、半透明はサポートしません。ピクセルは「完全に透明」か「完全に不透明」のどちらかです。

- 背景レイヤーを非表示にして書き出す → 透明背景GIF
- <Term id="anti-alias">アンチエイリアス</Term>の半透明部分は白に変換される

### 背景色を付ける

SNS投稿用には背景色を付けるのがおすすめ：

1. 最下層に背景レイヤーを作成
2. ドット絵に合う色で塗りつぶす
3. 書き出し

ダークグレー（#2a2a2a 等）は多くのドット絵と相性が良いです。

## 💡 Tips

### 再生速度の調整

GIFの再生速度がイメージと違う場合、<Term id="frame">フレーム</Term>時間を確認：

- ブラウザやSNSによって最低フレーム時間が異なる
- **20ms以下は無視される**ことが多い（ブラウザ依存）
- 安全のため **50ms以上** を推奨

### APNG という選択肢

GIFよりきれいなアニメーション画像が必要なら **APNG**（Animated PNG）もあります：

- 256色制限なし、フルカラー対応
- 半透明サポート
- ファイルサイズはGIFより大きめ
- 対応ブラウザが限られる（主要ブラウザは対応済み）

AsepriteからAPNGへの書き出しは、拡張子を `.png` にして **Export As** するだけです。

### プレビューで確認

書き出し前に **File > Preview** か、ブラウザにドラッグ＆ドロップして再生を確認しましょう。Asepriteのプレビューとブラウザの再生速度は微妙に異なることがあります。

### バッチ書き出し

複数ファイルを一括でGIFにしたい場合、Aseprite CLIが便利：

```bash
aseprite -b input.aseprite --scale 4 --save-as output.gif
```

フォルダ内の全ファイルを一括変換：

```bash
for f in *.aseprite; do
  aseprite -b "$f" --scale 4 --save-as "${f%.aseprite}.gif"
done
```
