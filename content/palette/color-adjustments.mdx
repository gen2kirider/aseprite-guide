## Adjustments とは

描き終わった後に「もう少し青っぽくしたい」「全体を暗くしたい」と思うことはよくあります。Asepriteの **Adjustments** 機能を使えば、<Term id="palette">パレット</Term>や個別ピクセルをいじらずに**色をまとめて調整**できます。

## 使い方

### メニューから開く

**Edit > Adjustments** から以下の調整機能にアクセスできます：

- **Brightness / Contrast** — 明るさとコントラスト
- **Hue / Saturation** — 色相と彩度
- **Color Curve** — トーンカーブ
- **Levels** — レベル補正
- **Invert** — 色の反転
- **Desaturate** — 彩度を0にしてグレースケール化

### 適用範囲

- **レイヤー全体** — <Term id="selection">選択範囲</Term>なしで実行すると、現在のレイヤーの現在フレーム全体に適用
- **選択範囲のみ** — 選択範囲を作ってから実行すると、その範囲だけに適用
- **全フレーム** — 全<Term id="cel">セル</Term>を選択してから実行すると、アニメーション全体に適用

## 各機能の詳細

### Brightness / Contrast

| パラメータ | 効果                         |
| ---------- | ---------------------------- |
| Brightness | +で明るく、-で暗く           |
| Contrast   | +でメリハリ強く、-でぼんやり |

- 描いたキャラが暗すぎた → Brightness を +20〜40
- 色が平坦に見える → Contrast を +10〜20

### Hue / Saturation

最もよく使う調整機能です。

| パラメータ | 効果                         |
| ---------- | ---------------------------- |
| Hue        | 色相を回転（-180° 〜 +180°） |
| Saturation | 彩度の増減                   |
| Lightness  | 明度の増減                   |

**活用例：**

- 赤い服を青い服に変えたい → Hue を +120° 程度
- 全体の色味を鮮やかに → Saturation を +20〜30
- 夜のシーンにしたい → Lightness を -30、Saturation を -10

> <Term id="index-color">インデックスカラー</Term>
> モードでは、この操作はパレット自体を変更します。RGBAモードではピクセルの色値を直接変えます。

### Color Curve

<Term id="rgba">RGB</Term>
の各チャンネル（赤・緑・青）ごとにトーンカーブを調整できます。

- **S字カーブ** — コントラスト強調
- **逆S字** — コントラスト低減
- **赤チャンネルだけ持ち上げ** — 全体に赤みを追加

### Invert（色の反転）

すべてのピクセルの色を反転します。白→黒、赤→シアンなど。

- ネガティブ表現
- 一部パーツだけ反転してカラーバリエーションに

### Desaturate（彩度除去）

カラーの<Term id="pixel-art">ドット絵</Term>をグレースケールに変換します。

複数のモードがあります：

- **Luminance** — 人間の目の感度に合わせた変換（推奨）
- **HSL** — HSLの明度を基準に変換
- **Average** — RGB平均値で変換

## 実践テクニック

### 配色バリエーションを素早く作る

1. キャラクターを描く
2. レイヤーを複製
3. 複製レイヤーに **Hue/Saturation** → Hue を回転
4. 別の色バリエーションが完成

### 時間帯の演出

同じ背景から昼・夕・夜バージョンを作る：

| 時間帯 | Hue | Saturation | Lightness |
| ------ | --- | ---------- | --------- |
| 昼     | 0   | 0          | 0         |
| 夕暮れ | +15 | -5         | -10       |
| 夜     | -20 | -20        | -35       |

### ダメージ表現の白フラッシュ

被ダメ時のフラッシュエフェクト：

1. キャラクターのフレームを複製
2. 複製に **Brightness** を +80〜100 に
3. Duration を 50ms に設定
4. 元のフレームに戻る

1フレームだけ白く光るダメージ演出になります。

## 💡 Tips

### プレビューで確認

Adjustments のダイアログにはリアルタイムプレビューがあります。スライダーを動かしながら<Term id="canvas">キャンバス</Term>の変化を確認して、ちょうどいい値を見つけましょう。

### Undo が効く

すべての Adjustments は `Ctrl+Z` で元に戻せます。気軽に試して、気に入らなければ戻しましょう。

### パレットの色数を保つには

RGBA モードで Adjustments を使うと、微妙に色が増えることがあります。色数を厳密に管理したい場合は、インデックスカラーモードで作業するか、適用後に **Sprite > Color Mode > Indexed** で一度変換して色を整理しましょう。
