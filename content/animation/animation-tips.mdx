## アニメーションの12原則

ディズニーが確立した「アニメーションの12原則」は<Term id="pixel-art">ドット絵</Term>にも応用できます。特に重要な原則を紹介します。

### スクワッシュ＆ストレッチ

物体が動くとき、潰れたり伸びたりする表現です。

- ジャンプの着地時 → キャラクターを横に1px広げ、縦に1px縮める
- ジャンプの上昇時 → 縦に1px伸ばす
- ボールのバウンド → 接地時に平たく、空中で丸く

> ドット絵は解像度が低いので、1〜2pxの変化で十分効果があります。やりすぎると崩れるので注意。

### 予備動作（アンティシペーション）

動く前に逆方向に少し動く表現：

1. ジャンプ前に少ししゃがむ（1〜2フレーム）
2. パンチ前に腕を引く（1フレーム）
3. 走り出す前に体を反対に傾ける

予備動作があると動きに「ため」が生まれ、自然に見えます。

### イージング

動きの速度を均一にせず、**始まりと終わりを緩やかに**する手法：

- **Ease In** — 最初はゆっくり、だんだん速く
- **Ease Out** — 最初は速く、だんだんゆっくり
- **Ease In-Out** — 両方

ドット絵では各フレームの移動量で表現します：

```
等速:       2px - 2px - 2px - 2px
Ease In:    1px - 1px - 2px - 4px
Ease Out:   4px - 2px - 1px - 1px
```

## フレーム数の目安

### 基本的なアクション

| アニメーション   | <Term id="frame">フレーム</Term>数 | フレーム時間 |
| ---------------- | ---------------------------------- | ------------ |
| 待機（アイドル） | 2〜4                               | 200〜300ms   |
| 歩き             | 4〜6                               | 100ms        |
| 走り             | 4〜6                               | 80ms         |
| ジャンプ         | 3〜5                               | 100ms        |
| 攻撃             | 3〜6                               | 80〜100ms    |
| 被ダメージ       | 2〜3                               | 100ms        |
| 死亡             | 4〜8                               | 120ms        |

> フレーム数は少ないほどキビキビした動きに、多いほど滑らかになります。ゲームのテンポに合わせて調整しましょう。

### フレーム時間の設定

Asepriteではフレームごとに表示時間を変えられます：

1. <Term id="timeline">タイムライン</Term>でフレームを右クリック
2. **Frame Properties** を選択
3. **Duration** を変更（ミリ秒単位）

## 実践テクニック

### アイドルアニメーション

キャラクターが何もしていないときの「待機アニメ」の作り方：

1. 通常ポーズを描く
2. 胸のあたりを1px上に動かしたフレームを追加（呼吸表現）
3. 2フレームの往復ループにする
4. フレーム時間を 250〜300ms に設定

たった2フレームでもキャラクターが生きている感じになります。

### 攻撃アニメーション

素早い攻撃アニメの構成：

1. **予備動作**（1フレーム）— 腕を引く / 体をひねる
2. **アクション**（1〜2フレーム）— 素早く振る。残像を1px追加すると迫力UP
3. **フォロースルー**（1〜2フレーム）— 腕が振り切った位置でゆっくり戻る

攻撃の瞬間は **80ms以下** の短いフレーム時間にすると、素早さが出ます。

### エフェクトアニメーション

ヒットエフェクトや爆発などのエフェクト：

- **3〜5フレーム**で十分
- 最初のフレームは**小さく明るく**、後半は**大きく薄く**
- 最後のフレームを1〜2px大きくして「広がり」を表現
- 白やハイライト色から始めると光っている感じに

## 💡 Tips

### ループの繋がりを確認

`Ctrl+P` でプレビュー再生。再生中も描画できるので、再生しながら微調整するとスムーズです。

### タグでアニメーションを管理

1つのファイルに複数のアニメーション（歩き・走り・攻撃など）を入れる場合、**タグ（Tag）**機能で管理します：

1. タイムラインでフレーム範囲を選択
2. 右クリック → **New Tag**
3. 名前（例: "walk", "run"）と色を設定
4. タグごとにループ再生できる

### <Term id="subpixel">サブピクセル</Term>アニメーション

1ピクセル未満の動きを表現するテクニック：

- 2フレームで1pxずつ交互に位置をずらす
- 人間の目には「半ピクセル動いた」ように見える
- 水面の揺れや、ゆっくりした呼吸に効果的

### フレームのコピーを活用

歩きアニメなどは左右対称の動きが多いです：

1. 前半のフレームを描く
2. フレームをコピー
3. **Edit > Flip Horizontal** で反転
4. 必要に応じて微調整

手間が半分になります。
