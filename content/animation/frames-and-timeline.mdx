## フレームとタイムライン

Asepriteでアニメーションを作るには<Term id="frame">フレーム</Term>と<Term id="timeline">タイムライン</Term>の理解が不可欠です。

### タイムラインの表示

`Tab` キーでタイムラインパネルの表示/非表示を切り替えます。

タイムラインは2つの軸で構成されています：

- **横軸** → <Term id="frame">フレーム</Term>（時間）
- **縦軸** → <Term id="layer">レイヤー</Term>

### フレームの操作

| 操作           | 方法            |
| -------------- | --------------- |
| フレームを追加 | `Alt+N`         |
| フレームを複製 | `Alt+D`         |
| フレームを削除 | `Alt+Delete`    |
| 次のフレームへ | `.`（ピリオド） |
| 前のフレームへ | `,`（カンマ）   |

### フレーム速度の設定

各フレームのヘッダーを右クリックすると **Frame Properties** が表示されます。

- **Duration**: フレームの表示時間（ミリ秒単位）
- デフォルトは 100ms（10fps 相当）

> **Tips:** ゲーム用<Term id="sprite">スプライト</Term>は 100ms（10<Term id="fps">fps</Term>）〜 66ms（15fps）が一般的です。

### プレビュー再生

`Enter` キーでアニメーションを再生/停止できます。再生速度はタイムラインの再生ボタン横で調整可能です。

### <Term id="onion-skin">オニオンスキン</Term>

前後のフレームを半透明で重ねて表示する機能です。タイムラインの🧅アイコンで有効にします。

アニメーションの動きを滑らかに繋げるのに非常に便利です。
