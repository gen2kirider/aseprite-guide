## はじめに

<Term id="pixel-art">ドット絵（ピクセルアート）</Term>
は1ピクセルずつ丁寧に色を置いていくアート手法です。Asepriteはこのドット絵制作に特化したツールで、最小限の機能で最大の表現ができるように設計されています。

## キャンバスサイズの決め方

ドット絵で最も重要な最初の選択が<Term id="canvas">キャンバス</Term>サイズです。

### よく使われるサイズ

| 用途               | サイズ           | 補足                                     |
| ------------------ | ---------------- | ---------------------------------------- |
| キャラクター（小） | 16×16            | ファミコン風                             |
| キャラクター（中） | 32×32            | SFC〜GBA風                               |
| キャラクター（大） | 64×64            | 高精細ドット絵                           |
| アイコン           | 16×16 / 32×32    | ゲームUIやSNS用                          |
| 背景タイル         | 16×16 / 32×32    | <Term id="tilemap">タイルマップ</Term>用 |
| イラスト           | 128×128〜320×240 | 作品として仕上げる場合                   |

> 迷ったら **32×32** から始めるのがおすすめ。小さすぎず大きすぎず、練習に最適なサイズです。

### 新規作成の手順

1. **File > New** を開く（`Ctrl+N`）
2. Width と Height にサイズを入力
3. Color Mode は **Indexed** か **RGBA** を選択
4. **OK** で作成

## きれいなドットを打つコツ

### 1ピクセル線を意識する

ドット絵の線は基本的に **1ピクセル幅** です。斜め線を描くときは「階段状」になりますが、段差のパターンを均一にすると滑らかに見えます。

- ❌ 悪い例: 1-1-2-1-3-1（段差がバラバラ）
- ✅ 良い例: 1-2-1-2-1-2（段差が均一）
- ✅ 良い例: 1-1-2-1-1-2（パターンが規則的）

### ジャギーを消す

斜め線の「ガタガタ」をジャギーと呼びます。これを滑らかにするテクニック：

1. 段差が不均一な部分を見つける
2. ピクセルを1つずつ移動して段差を均一にする
3. 必要に応じて中間色のピクセルを置く（<Term id="anti-alias">アンチエイリアス</Term>）

> ドット絵では**あえてアンチエイリアスを使わない**スタイルも人気です。特に小さいサイズ（16×16など）ではくっきりした線の方がきれいに見えます。

### アウトライン

キャラクターの<Term id="outline">アウトライン</Term>（輪郭線）の描き方は主に2種類：

- **黒アウトライン** — くっきりとした印象。レトロゲーム風
- **カラーアウトライン** — パーツごとに暗い色でアウトライン。柔らかい印象

## 基本的な描画の流れ

### ステップ1: シルエット

まず1色でキャラクターの形を描きます。この段階でプロポーションを決めます。

### ステップ2: 色分け

各パーツのベース色を塗ります。バケツツール（`G`）が便利です。

### ステップ3: 影とハイライト

光源の方向を決めて、影色とハイライトを塗ります。

### ステップ4: 仕上げ

- アウトラインの色を調整
- 不要なピクセルを削除
- 全体のバランスを確認

## 💡 Tips

### Zoom機能を活用する

- `Ctrl++` / `Ctrl+-` でズームイン・アウト
- `Ctrl+0` で等倍表示
- 描くときは **400〜800%** くらいにズームして作業
- 時々等倍に戻して全体のバランスを確認する癖をつける

### グリッド表示

**View > Show Grid** でピクセルグリッドを表示できます。16×16のキャラクターを描くときなど、境界が見やすくなります。

### 対称ツールで時短

左右対称のキャラクターを描くときは対称ツールが便利：

1. ツールバーの対称オプションをON
2. 水平対称を選択
3. 片側だけ描けば反対側が自動で反映される

### ピクセルパーフェクトモード

鉛筆ツールのオプションに **Pixel Perfect** があります。ONにすると斜め線を描いたときに余分なピクセルが自動的に除去され、きれいな1ピクセル線が引けます。
