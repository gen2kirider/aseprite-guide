## Asepriteの画面構成

Asepriteを初めて起動すると、いくつかのパネルが表示されます。それぞれの役割を理解しましょう。

### メインキャンバス

画面中央に表示される領域が**メイン<Term id="canvas">キャンバス</Term>**です。ここで<Term id="pixel-art">ドット絵</Term>を描きます。

- マウスホイールで**ズーム**できます
- `Space` を押しながらドラッグで**パン**（スクロール）できます

### ツールバー

画面左側にあるアイコン列が**ツールバー**です。よく使うツール：

| ツール   | ショートカット | 説明              |
| -------- | -------------- | ----------------- |
| 鉛筆     | `B`            | 1ピクセルずつ描画 |
| 消しゴム | `E`            | ピクセルを削除    |
| スポイト | `I`            | 色を拾う          |
| バケツ   | `G`            | 塗りつぶし        |
| 選択     | `M`            | 矩形選択          |

### カラーパレット

画面下部に表示されるのが**<Term id="palette">カラーパレット</Term>**です。使用する色を選択・管理できます。

### <Term id="timeline">タイムライン</Term>

画面下部の`Tab`キーで表示/非表示を切り替えられるパネルです。アニメーションの<Term id="frame">フレーム</Term>管理に使います。

## 新規ファイルの作成

`Ctrl+N`（Mac: `Cmd+N`）で新規<Term id="sprite">スプライト</Term>を作成できます。

1. **幅・高さ**を入力（例: 32×32, 64×64）
2. **カラーモード**を選択（通常は <Term id="rgba">RGBA</Term>）
3. **背景色**を選択（透明推奨）
4. 「OK」をクリック

> **Tips:** ドット絵は小さいサイズから始めるのがオススメです。16×16 や 32×32 で練習しましょう。
