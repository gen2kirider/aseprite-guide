## ブラシの基本

Asepriteの<Term id="brush">ブラシ</Term>は単なるペンツールではありません。形状やサイズ、描画モードをカスタマイズすることで作業効率が大幅に上がります。

### ブラシサイズの変更

ブラシサイズを変更する方法はいくつかあります：

- 画面上部のオプションバーで数値を入力
- マウスホイール（ツール選択中）
- `+` / `-` キー

> ドット絵ではほとんどの場合 **1px** で描きます。2px以上は太い線や塗りつぶし用と考えましょう。

### ブラシの形状

| 形状          | 用途               |
| ------------- | ------------------ |
| 丸 (Circle)   | 標準的な描画       |
| 四角 (Square) | タイルの塗りに便利 |
| 線 (Line)     | 斜線パターンの作成 |
| カスタム      | 自作ブラシ         |

## カスタムブラシの作り方

Asepriteの強力な機能の一つが**カスタムブラシ**です。

### 選択範囲からブラシを作成

1. キャンバス上にパターンを描く（例: 草のパターン）
2. 選択ツール（`M`）でそのパターンを選択
3. **Edit > New Brush**（`Ctrl+B`）を実行
4. ブラシとして使えるようになる

これで描画するたびにそのパターンがスタンプのように配置されます。

### ブラシの活用例

- **草や地面のテクスチャ** — ランダムなドットパターンをブラシ化
- **レンガ模様** — 1ユニット分を描いてブラシ化
- **木の葉** — 葉のかたまりをブラシにして効率よく配置
- **装飾パターン** — 繰り返し模様の一部をブラシ化

## 描画モード

ブラシには「どのように色を置くか」を決める描画モードがあります。

### よく使うモード

- **Normal** — 通常の描画。上に重ねて塗る
- **Behind** — 既に色があるピクセルの「後ろ」に塗る。塗り残し埋めに便利
- **Eraser** — 透明色で描画（消しゴムと同じ効果）

### ブレンドモード付き描画

Ink の設定でさらに高度な描画が可能です：

| Ink               | 効果                                                 |
| ----------------- | ---------------------------------------------------- |
| Simple Ink        | 通常描画                                             |
| Alpha Compositing | 半透明で描画                                         |
| Shading           | <Term id="palette">パレット</Term>上で隣の色にシフト |
| Lock Alpha        | 透明部分には描画しない                               |

## Shading Inkの使い方

**Shading Ink** はドット絵制作で非常に便利な機能です。

### 仕組み

塗った場所の色が、パレット上で**1つ隣の色**に変わります：

- 左クリック → パレットの右方向（明るい方向）にシフト
- 右クリック → パレットの左方向（暗い方向）にシフト

### 活用シーン

1. パレットに明→暗の順で色を並べておく
2. Shading Inkを選択
3. 明るくしたい部分を左クリックで塗る
4. 暗くしたい部分を右クリックで塗る

> パレットの色順が重要です。Shading Inkを使う前に、パレットをベース色→影色→ハイライトの順に整理しておきましょう。

## 💡 Tips

### ブラシのプレビュー

描画前にブラシがどのように見えるかを確認するには、ツールオプションの **Preview** を有効にします。カーソルにブラシの形状が表示されます。

### よく使うブラシをショートカットに

カスタムブラシは保存して何度でも呼び出せます。よく使うブラシは **ショートカットに登録** しておくと効率的です。

### ディザリングブラシ

2色を交互に配置する「<Term id="dithering">ディザリング</Term>」をブラシで実現できます：

1. 2×2のチェッカーパターンを描く
2. カスタムブラシとして登録
3. 塗ると自動でディザリングパターンが描画される

グラデーションの表現や、限られた色数での中間色表現に活用できます。

### 1px ブラシに戻すショートカット

カスタムブラシを使った後に標準の1pxブラシに戻すには、ブラシサイズを `1` にするだけです。カスタムブラシがリセットされて通常のドット描画に戻ります。
