## ショートカット一覧

よく使うショートカットを覚えて作業効率をアップしましょう。

### ファイル操作

| 操作         | Windows/Linux  | macOS         |
| ------------ | -------------- | ------------- |
| 新規作成     | `Ctrl+N`       | `Cmd+N`       |
| 開く         | `Ctrl+O`       | `Cmd+O`       |
| 保存         | `Ctrl+S`       | `Cmd+S`       |
| 別名で保存   | `Ctrl+Shift+S` | `Cmd+Shift+S` |
| エクスポート | `Ctrl+Shift+E` | `Cmd+Shift+E` |

### ツール

| ツール   | ショートカット |
| -------- | -------------- |
| 鉛筆     | `B`            |
| 消しゴム | `E`            |
| スポイト | `I`            |
| バケツ   | `G`            |
| 直線     | `L`            |
| 矩形     | `U`            |
| 楕円     | `Shift+U`      |
| 矩形選択 | `M`            |
| 投げ縄   | `Q`            |
| 魔法の杖 | `W`            |
| 移動     | `V`            |
| ズーム   | `Z`            |
| ハンド   | `H`            |

### 編集

| 操作         | ショートカット |
| ------------ | -------------- |
| 元に戻す     | `Ctrl+Z`       |
| やり直し     | `Ctrl+Y`       |
| コピー       | `Ctrl+C`       |
| ペースト     | `Ctrl+V`       |
| 全選択       | `Ctrl+A`       |
| 選択解除     | `Ctrl+D`       |
| 選択範囲反転 | `Ctrl+Shift+I` |
| 切り取り     | `Ctrl+X`       |

### アニメーション

| 操作         | ショートカット |
| ------------ | -------------- |
| 再生/停止    | `Enter`        |
| 次のフレーム | `.`            |
| 前のフレーム | `,`            |
| 新規フレーム | `Alt+N`        |
| フレーム複製 | `Alt+D`        |
| フレーム削除 | `Alt+Delete`   |

### 表示

| 操作                                        | ショートカット |
| ------------------------------------------- | -------------- |
| <Term id="timeline">タイムライン</Term>表示 | `Tab`          |
| ズームイン                                  | `+` / `Ctrl+=` |
| ズームアウト                                | `-` / `Ctrl+-` |
| グリッド表示                                | `Shift+G`      |
| ピクセルグリッド                            | `Alt+G`        |
| 全画面表示                                  | `F11`          |

> **Tips:** `Edit` → `Keyboard Shortcuts` から自分好みにカスタマイズできます。
