## カスタマイズの基本

Asepriteはほぼすべての操作にショートカットを割り当てられます。自分の作業スタイルに合わせてカスタマイズすることで、描画速度が大幅に向上します。

### 設定画面を開く

**Edit > Keyboard Shortcuts**（`Ctrl+Alt+Shift+K`）

設定画面では以下のカテゴリに分かれています：

- **Menu** — メニュー項目のショートカット
- **Commands** — 各種コマンド
- **Tools** — ツール切り替え
- **Actions** — アクション

## おすすめカスタマイズ

### 左手だけで操作できる配置

右手でペンタブ、左手でキーボードという作業スタイルで使いやすい配置：

| キー  | 割り当て           | 理由                 |
| ----- | ------------------ | -------------------- |
| Q     | スポイト           | 左手小指で押しやすい |
| W     | 消しゴム           | 頻繁に使うツール     |
| A     | 1つ前の色          | 色の切り替え         |
| S     | 1つ後の色          | 色の切り替え         |
| D     | ブラシサイズ+1     | サイズ調整           |
| F     | ブラシサイズ-1     | サイズ調整           |
| Z     | Undo               | 定番                 |
| X     | 前景色/背景色 切替 | デフォルトのまま     |
| Space | パン（手のひら）   | デフォルトのまま     |

### フレーム操作

アニメーション作業用：

| キー            | 割り当て       |
| --------------- | -------------- |
| `,`（カンマ）   | 前のフレーム   |
| `.`（ピリオド） | 次のフレーム   |
| `/`             | 再生/停止      |
| `Shift+,`       | 最初のフレーム |
| `Shift+.`       | 最後のフレーム |

## カスタマイズ手順

### 新しいショートカットを追加

1. Keyboard Shortcuts を開く
2. 変更したい項目を検索
3. 項目をクリック
4. **Add Shortcut** ボタンを押す
5. 割り当てたいキーを押す
6. **OK** で確定

### 既存のショートカットを変更

1. 変更したい項目を見つける
2. 既存のショートカットの横の **×** で削除
3. 新しいキーを割り当てる

### 競合の確認

既に別の機能に使われているキーを割り当てようとすると、**競合警告**が出ます。

- 競合を上書きしてもOK（元の機能のショートカットが外れる）
- 1つのキーに複数の機能は割り当てられない

## プリセットの管理

### 設定のエクスポート

**Edit > Keyboard Shortcuts** の画面で **Export** ボタンを押すと、現在の設定をファイルに保存できます。

### 設定のインポート

同じ画面の **Import** ボタンで、保存した設定を読み込めます。

> PCを買い替えたときや、複数のPCで同じ環境を使いたいときに便利です。

### デフォルトに戻す

**Reset** ボタンで初期状態に戻せます。カスタマイズしすぎて混乱したら、一度リセットしてやり直しましょう。

## 💡 Tips

### ペンタブのボタンも活用

ペンタブのサイドボタンにもショートカットを割り当てると便利です（ペンタブのドライバ設定で）：

| ボタン        | おすすめ割り当て  |
| ------------- | ----------------- |
| サイドボタン1 | Undo（`Ctrl+Z`）  |
| サイドボタン2 | スポイト（`Alt`） |
| サイドボタン3 | ブラシサイズ変更  |

### マウスホイールの活用

デフォルトでマウスホイールはズームに割り当てられています。変更したい場合：

- `Ctrl+ホイール` → ブラシサイズ変更
- `Shift+ホイール` → フレーム送り

Keyboard Shortcuts の **Mouse Wheel** セクションで設定できます。

### よく使う操作を覚える

カスタマイズの前に、まずデフォルトのショートカットで**特に便利なもの**を覚えましょう：

1. `B` — ブラシツール
2. `E` — 消しゴム
3. `G` — 塗りつぶし
4. `M` — 選択ツール
5. `I` — スポイト
6. `Z` — ズーム
7. `Ctrl+Z` — Undo
8. `Ctrl+D` — 選択解除
9. `Tab` — タイムライン表示切替
10. `F3` — <Term id="onion-skin">オニオンスキン</Term>切替

これらをまず身体で覚えてから、不便な部分だけカスタマイズするのが効率的です。
