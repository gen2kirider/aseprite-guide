## レイヤーの基本

<Term id="layer">レイヤー</Term>
は透明なシートを重ね合わせるイメージです。パーツごとにレイヤーを分けることで編集が楽になります。

### レイヤーの種類

- **Normal（通常レイヤー）**: 一般的なピクセル描画用
- **Background（背景レイヤー）**: 透明にできない最下層レイヤー
- **Group（グループ）**: レイヤーをまとめるフォルダ
- **<Term id="tilemap">Tilemap</Term>（タイルマップ）**: タイルベースの描画用
- **Reference（参考資料）**: 書き出しに含まれない参考画像用

### レイヤー操作

| 操作           | ショートカット                      |
| -------------- | ----------------------------------- |
| 新規レイヤー   | `Shift+N`                           |
| レイヤー削除   | レイヤーパネルで右クリック → Delete |
| レイヤーの複製 | 右クリック → Duplicate              |
| レイヤーの結合 | `Ctrl+E`（下のレイヤーと結合）      |
| 表示/非表示    | 👁アイコンをクリック                |
| ロック         | 🔒アイコンをクリック                |

### <Term id="blend-mode">ブレンドモード</Term>

レイヤーのブレンドモードを変更して、合成方法を変えられます：

- **Normal**: 通常の重ね合わせ
- **Multiply**: 暗い色を強調（影に便利）
- **Screen**: 明るい色を強調（光に便利）
- **Addition**: 色を加算（発光表現に）

> **Tips:** キャラクターを描くときは「線画」「塗り」「影」「ハイライト」のように役割別にレイヤーを分けましょう。
