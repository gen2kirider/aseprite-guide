## <Term id="tileset">タイルセット</Term>設計のコツ

### タイルサイズの選び方

| サイズ | 用途                   | 特徴                               |
| ------ | ---------------------- | ---------------------------------- |
| 8×8    | レトロゲーム風         | 制約が強いがチップ数が少なくて済む |
| 16×16  | 一般的なドット絵ゲーム | 最もバランスが良い                 |
| 32×32  | 高解像度ドット絵       | 表現の幅が広いがチップ数が増える   |

> 迷ったら **16×16** から始めるのがおすすめです。

### 基本タイルの構成

地形<Term id="tile">タイル</Term>（草地など）を作るために最低限必要なパーツ：

- **中央**（繰り返しパターン）— 1枚
- **上下左右の端**— 4枚
- **四隅（外角）**— 4枚
- **四隅（内角）**— 4枚

合計 **13枚** で基本的な地形が表現できます。

## 繰り返しパターンの作り方

### シームレスタイル

隣り合うタイルの境界が目立たないように描くテクニック：

1. 中央にパターンを描く
2. **Tile Mode** をONにする（View > Tiled Mode > Both Axes）
3. キャンバスの外にも同じタイルが並んで表示される
4. 境界部分を描き足して繋がりを自然にする

> Tiled Mode はタイルセット作りの必須機能です。常にONにして描きましょう。

### バリエーションタイル

同じ草地でも1種類だけだと単調になります：

1. 基本パターンを描く
2. タイルをコピー
3. 小石を追加、草の形を少し変えるなど微妙な変化を加える
4. 3〜4種類のバリエーションを作る

マップ上でランダムに配置すると、自然な見た目になります。

## 端（エッジ）タイルの描き方

### 外角タイル

地形の外側の角。草地が途切れて土が見える部分：

1. 基本タイルを複製
2. 角の部分を削って下の地形の色にする
3. 境界線を描く（1〜2pxの暗い線）

### 内角タイル

L字型に地形が繋がる部分の内側の角：

1. 端タイルを2枚組み合わせたような形
2. 外角より描くのが難しいので、先に外角を完成させてから

### 段差の描き方

横から見た段差（崖）の表現：

1. 上面タイル — 草地の表面
2. 壁面タイル — 土や岩の断面
3. 下端タイル — 影を落とす

壁面は上下に繰り返せるように描くと、高さの違う崖に対応できます。

## Asepriteでの効率的な作業

### タイルマップモードの活用

1. **New <Term id="tilemap">Tilemap</Term> Layer** でタイルマップ専用レイヤーを作成
2. タイルパレットにタイルが自動登録される
3. タイルを描く → 自動でパレットに追加
4. パレットからタイルを選んで配置

### グリッド表示

**View > Grid > Grid Settings** でグリッドをタイルサイズに合わせます：

1. Width / Height をタイルサイズに設定（例: 16×16）
2. **View > Grid > Show Grid**（`Ctrl+'`）でグリッド表示

### スナップ

**View > Snap to Grid** をONにすると、描画やコピーがグリッドに吸着します。

## 💡 Tips

### タイルの確認方法

描いたタイルが実際にどう見えるか確認するには：

1. 新しいレイヤーを作成
2. タイルをコピーして並べてみる
3. 不自然な繋がりがないか確認
4. Tiled Mode でリアルタイムに確認するのが最も効率的

### 色数を抑える

<Term id="palette">パレット</Term>の色数を抜えた方が統一感が出ます：

- 1つの地形につき **4〜8色** が目安
- 暗→中→明→ハイライトの段階で色を選ぶ
- 地形間で共通の色を使うと調和する

### RPG向けタイルの優先順位

RPGのマップを作るなら、以下の順で作ると効率的：

1. **草地**（最も面積が広い）
2. **道**（プレイヤーが歩く場所）
3. **水辺**（端タイルのパターンが参考になる）
4. **壁・建物**
5. **装飾**（花、石、木など）

### 書き出し

完成したタイルセットをゲームで使うには：

1. タイルパレットを画像として書き出す
2. **File > Export As** で PNG保存
3. ゲームエンジンで読み込み、グリッドサイズを指定

Godot の **TileMap** ノードや Unity の **Tilemap** コンポーネントで直接使えます。
